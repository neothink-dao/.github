## 🛠️ **Technology Stack**

- **Frontend**: Next.js 15.5, React 19.1, TypeScript 5.9.2
- **AI Framework**: Vercel AI SDK 5.0 with multi-modal capabilities

- **Styling**: Tailwind CSS 4.x, shadcn/ui Platform
